# Лог генератора — PWA

Прогресивна веб-апка для обліку часу роботи генератора. Працює офлайн,
ставиться на робочий стіл Android як звичайна апка, дані зберігаються в
IndexedDB (переживають перезавантаження, оновлення, перезапуск телефону —
зникають лише при видаленні апки або ручному очищенні).

## Файли
- `index.html` — сама апка
- `manifest.json` — опис апки (іконка, назва, режим)
- `sw.js` — service worker (офлайн-кеш)
- `icons/` — іконки 192 та 512 px

## Важливо: чому не можна просто відкрити файл
Service worker і встановлення PWA працюють ТІЛЬКИ через `https://` або
`http://localhost`. Якщо відкрити `index.html` напряму (file://), апка
запуститься і дані зберігатимуться, але не буде ні офлайн-кешу, ні кнопки
"встановити". Тому файли треба віддати через веб-сервер.

## Варіант А — найпростіший: GitHub Pages (безкоштовно, https з коробки)
1. Створи репозиторій на GitHub.
2. Залий туди вміст папки `generator-pwa` (index.html має бути в корені).
3. Settings → Pages → Source: гілка `main`, папка `/root` → Save.
4. За хвилину отримаєш адресу виду `https://твій-нік.github.io/repo/`.
5. Відкрий її в Chrome на Android → меню (⋮) → "Додати на головний екран" /
   "Встановити застосунок".

## Варіант Б — локальний сервер (перевірити на компʼютері)
У папці з файлами:
    python3 -m http.server 8080
Відкрий http://localhost:8080 у Chrome. Для встановлення на телефон цей
варіант не годиться (localhost телефону != компʼютер), але зручно тестувати.

## Варіант В — будь-який хостинг
Залий папку на Netlify (перетягни папку на app.netlify.com/drop), Vercel,
Cloudflare Pages чи власний сервер з https. Усі дають https безкоштовно.

## Встановлення на Android
Відкрив адресу в Chrome → ⋮ → "Встановити застосунок" / "Додати на головний
екран". Зʼявиться іконка, апка відкривається на весь екран без адресного
рядка й повністю працює без інтернету.

## Бекап даних
- **Експорт CSV** — звіт для Excel.
- **Бекап JSON** — повна копія бази; поклади файл кудись у надійне місце.
- **Імпорт JSON** — відновлення з бекапу (замінює поточні дані).

Раджу зрідка робити бекап JSON — IndexedDB надійний, але це твоя єдина
страховка, якщо телефон загубиться або апку знесе.

## Якщо захочеш APK
Цю ж папку можна без змін загорнути в нативний APK через Capacitor або
PWABuilder (pwabuilder.com приймає URL твоєї PWA і віддає готовий APK).
Код міняти не треба.
